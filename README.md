# 🚀 Galaxy Force

[![Python](https://img.shields.io/badge/Python-3.10+-blue)](https://www.python.org/)
[![Pygame](https://img.shields.io/badge/Pygame-2.1+-green)](https://www.pygame.org/news)

**Galaxy Force** é um jogo de ação em 2D com rolagem vertical, feito em Python com Pygame. Controle sua nave, enfrente ondas de inimigos, colete power-ups e alcance a maior pontuação!

---

## 🎮 Como Rodar

### Pré-requisitos
- Python 3.10+  
- Pygame 2.1+

Instale o Pygame via terminal:

```bash
pip install pygame
```
Como posso jogar o Galaxy Force?
Clone o repositório:

```bash
git clone https://github.com/pillowzin/Galaxy-Force.git
```
Entre na pasta do projeto:
```bash
cd Galaxy-Force
```
Execute o jogo:

```bash
python main.py
```
⚠️ Mantenha o terminal no diretório do projeto para que o jogo encontre todos os arquivos.

⌨️ Controles
Mouse → Movimentar a nave
Tecla Q → Atirar
Esc → Sair do jogo

🌌 Funcionalidades
Rolagem vertical fluida
Variados tipos de inimigos e padrões de ataque
Sistema de pontuação
Efeitos visuais (estrelas, explosões)
Música e efeitos sonoros
Frenético!

🗂 Estrutura do Projeto
```csharp
Galaxy-Force/
│
├─ main.py          # Arquivo principal do jogo
├─ const.py         # Constantes do jogo
├─ objects.py       # Classes de nave, inimigos e tiros
├─ gameStates.py    # Estados do jogo (menu, jogando, game over)
├─ enemyFactory.py  # Criação de inimigos por fase
├─ effects.py       # Efeitos visuais (estrelas, explosões)
├─ misc/            # Ícone e fonte
├─ sprites/         # Imagens e spritesheets
└─ sounds/          # Música e efeitos sonoros
```
